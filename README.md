# Analysis_of_an_Equity_Investment_Portfolio_-2016-2026-
This project analyzes the historical performance, risk profile, and efficiency of a diversified equity investment portfolio composed of Apple (AAPL), Microsoft (MSFT), Amazon (AMZN), and JPMorgan Chase (JPM), using the Vanguard S&amp;P 500 ETF (VOO) as the benchmark.  Using historical data from 2016 to 2026.
